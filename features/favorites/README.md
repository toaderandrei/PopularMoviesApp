# :features:favorites

Favorites management feature module. Displays saved favorite movies and TV shows with sync status indicators and remote sync capability.

## Key Classes
- `FavoritesViewModel` / `FavoritesUiState` - Favorites list with movies and TV shows
- `FavoritesDetailsViewModel` / `FavoritesDetailsRoute` - Favorite detail view

## Navigation Routes
- `favorites` - Main favorites screen (bottom nav tab)

## Dependencies
- `core:domain`, `core:models`, `core:ui`, `core:resources`, `core:shared`
