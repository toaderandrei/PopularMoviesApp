# :features:search

Search feature module. Provides debounced search across movies and TV shows with tabbed results display.

## Key Classes
- `SearchViewModel` / `SearchUiState` - Search state with debounced query handling
- `SearchRoute` / `SearchScreen` - Search UI with results tabs

## Navigation Routes
- `search` - Search screen (top bar action)

## Dependencies
- `core:domain`, `core:models`, `core:ui`, `core:resources`, `core:common`
